# Proyek Membuat Sistem Rekomendasi 🎯

Selamat datang di repositori **Proyek Membuat Sistem Rekomendasi**! Proyek ini bertujuan untuk mengembangkan sistem rekomendasi menggunakan teknik machine learning. Di sini, Anda akan menemukan berbagai skrip dan notebook yang digunakan untuk membangun dan menguji sistem rekomendasi.

---

## Fitur Utama ✨

- **Sistem Rekomendasi**: Membangun model rekomendasi berdasarkan data pengguna dan item.
- **Analisis Data**: Menggunakan teknik analisis untuk memahami pola dalam data.
- **Visualisasi**: Menyediakan visualisasi untuk hasil rekomendasi.

---

## Sumber Data 📥

Database untuk proyek ini tidak dapat dimasukkan ke dalam GitHub karena ukuran yang besar. Anda dapat mengakses database melalui tautan berikut:

- [Akses Database](https://drive.google.com/file/d/1_fUjMTz4uT4Ldi3R1h5T2Lex9RTxcb1S/view?usp=sharing)

Silakan unduh dan simpan database di direktori proyek Anda sebelum menjalankan skrip.

---

## Setup Environment 🛠️

Untuk memulai, Anda perlu menginstal semua dependensi yang diperlukan. Anda dapat melakukannya dengan menjalankan perintah berikut:

```bash
pip install -r requirements.txt
```

---

## Setup Environment - Shell/Terminal 💻

Ikuti langkah-langkah berikut untuk menyiapkan lingkungan proyek Anda:

```bash
mkdir proyek_sistem_rekomendasi
cd proyek_sistem_rekomendasi
pipenv install
pipenv shell
pip install -r requirements.txt
```

---

## Menjalankan Notebook Jupyter 📓

Setelah semua terinstal, Anda dapat menjalankan notebook Jupyter dengan perintah berikut:

```bash
jupyter notebook
```

Kemudian, buka file `Submission_Terapan_Rekomendasi.ipynb` untuk mulai menganalisis data dan mengembangkan sistem rekomendasi.

---

## Struktur Proyek 📁

- `Submission_Terapan_Rekomendasi.ipynb`: Notebook untuk pengembangan sistem rekomendasi.
- `submission_terapan_rekomendasi.py`: Skrip utama untuk menjalankan sistem rekomendasi.
- `requirements.txt`: Daftar semua dependensi yang diperlukan untuk proyek ini.
